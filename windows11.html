Windows 11 Exclusive

Windows 11 Exclusive Section Allows You To Control The Following:

Disable VBS:

Disables Virtualization-Based Security (VBS) for advanced security features.

Move Taskbar to The Left:

Moves the taskbar from the default bottom position to the left side of the screen.

Disable Snap Assist:

Disables Snap Assist, which automatically arranges open windows when dragging.

Disable Widgets:

Disables the Widgets feature on the Windows 11 taskbar.

Disable Chat:

Disables the Chat feature on the Windows 11 taskbar.

Enable Legacy Volume Slider:

Enables the legacy volume slider in the system tray.

Disable Show More Options:

Hides additional options in various Windows settings.

Disable TPM Check:

Disables the requirement for Trusted Platform Module (TPM) during Windows installation.

Enable Files Compact Mode:

Enables Compact Mode for file browsing with smaller thumbnails.

Disable Stickers:

Disables the Stickers feature in the Windows 11 messaging app.

Disable Edge Discover Bar:

Disables the Discover Bar feature in Microsoft Edge.

Disable CoPilot AI:

Disables the CoPilot AI feature in Windows for assistance with code writing.
