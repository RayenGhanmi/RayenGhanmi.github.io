Telemetry

Telemetry Section Allows You To Control The Following:

Enhance Privacy:

Toggle various privacy-enhancing settings to protect your personal information.

Learn more about what's being disabled

Disable SMBv1:

Disables Server Message Block version 1 (SMBv1) for enhanced security.

Disable SMBv2:

Disables Server Message Block version 2 (SMBv2) for enhanced security.

Disable SMBv3:

Disables Server Message Block version 3 (SMBv3) for enhanced security.

Disable Telemetry Services:

Turns off telemetry services to prevent the collection and transmission of usage data by the operating system.

Disable Edge Telemetry:

Disables telemetry data collection in Microsoft Edge.

Disable Visual Studio Telemetry:

Disables telemetry data collection in Visual Studio.

Disable Nvidia Telemetry:

Disables telemetry data collection by Nvidia drivers.

Disable Chrome Telemetry:

Disables telemetry data collection in Google Chrome.

Disable Firefox Telemetry:

Disables telemetry data collection in Mozilla Firefox.
