    Usage

    Launching the Application

    1. Open the Start menu and search for RyTuneX.
    2. Click on the RyTuneX application.
    3. Accept the UAC prompt and you should be good to go.

    Navigating the Application

    • Home Page: This is the starting point for navigating through RyTuneX. It includes real-time usage statistics and a count of installed apps and services.
    • Optimize: Access various system optimization features to improve performance. This includes performance tweaks, and more.
    • Repair: Access various diagnostic and repair options to fix common system issues and ensure your system runs smoothly.
    • Debloat: Uninstall unwanted UWP apps and remove temporary files to free up system resources and improve performance.
    • Privacy: Disable telemetry services and enhance privacy settings to protect your personal data and reduce background data collection.
    • Features: Control your Windows features, enabling or disabling specific functionalities to customize your system according to your needs.
    • Networking: Change DNS server settings and reset DNS to default values. This helps in managing your network settings for better performance and security.
    • Device: View detailed information about the operating system, CPU, GPU, RAM, and disk drives. This section provides comprehensive hardware and system information.
    • Security: Preview the current state of different system security features to keep your system as secure as possible.
    • Settings: Configure application settings, including theme and language preferences, to personalize your RyTuneX experience.

    Performing Common Tasks

    Uninstalling UWP Apps

    1. Navigate to the Debloat page.
    2. Select the apps you want to uninstall from the list.
    3. Click the Uninstall button to remove the selected apps.

    Changing DNS Server

    1. Navigate to the Networking page.
    2. Select a DNS server from the dropdown list.
    3. Click the Apply DNS button to change the DNS server.

    Extracting Drivers

    1. Navigate to the Device page.
    2. Click the Extract Drivers button to extract drivers to a specified folder.

    Optimizing System Performance

    1. Navigate to the Optimize page.
    2. Toggle the performance tweaks you want and they should be applied automatically.

    Running Diagnostics and Repairs

    1. Navigate to the Repair page.
    2. Click the Check/Diagnose button to start the diagnostic or repair process you want.

    Enhancing Privacy Settings

    1. Navigate to the Privacy page.
    2. Toggle the privacy settings you want to enable or disable.

    Controlling Windows Features

    1. Navigate to the Features page.
    2. Toggle the Windows features you want to enable or disable.

    Viewing System Information

    1. Navigate to the Device page.
    2. View detailed information about the operating system, CPU, GPU, RAM, and disk drives.

    Previewing Security Features

    1. Navigate to the Security page.
    2. Preview the current state of different system security features.

    Configuring Application Settings

    1. Navigate to the Settings page.
    2. Configure application settings, including theme and language preferences.

    Reverting Applied Changes

    1. Navigate to the Settings page.
    2. Scroll down to the Revert Changes section.
    3. Click the Revert button to undo all changes applied by RyTuneX to the system.
