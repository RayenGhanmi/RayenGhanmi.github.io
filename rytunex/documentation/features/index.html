    Features

    System Optimization

    Basic System Optimization

    • Menu Show Delay: Adjust the delay before menus are shown.
    • Mouse Hover Time: Adjust the time before hover actions are triggered.
    • Auto Complete: Enable or disable auto-complete features.
    • Crash Dump: Enable or disable crash dump creation.
    • Remote Assistance: Enable or disable remote assistance.
    • Window Shake: Enable or disable the Aero Shake feature.
    • Copy Move Context Menu: Enable or disable copy/move context menu options.
    • Task Timeouts: Adjust task timeouts for better performance.
    • Low Disk Space Checks: Enable or disable low disk space checks.
    • Link Resolve: Enable or disable link resolution.
    • Service Timeouts: Adjust service timeouts for better performance.
    • Remote Registry: Enable or disable remote registry access.
    • File Extensions and Hidden Files: Show or hide file extensions and hidden files.

    Advanced System Optimization

    • System Profile: Adjust system profile settings for performance.
    • GPU and Priority Settings: Optimize GPU and process priority settings.
    • Frame Server Mode: Enable or disable frame server mode.
    • Low Latency GPU Settings: Optimize GPU settings for low latency.
    • Non Best Effort Limit: Adjust non-best effort limits for better performance.
    • Disable SysMain: Disable the SysMain service to reduce unnecessary disk usage.
    • Disable NTFS Timestamp: Disable NTFS timestamp updates to improve file system performance.
    • Enable Gaming Mode: Optimize system settings for gaming performance.
    • Exclude Drivers from Windows Updates: Prevent Windows from updating drivers automatically.

    Other System Optimization

    • System Restore: Enable or disable system restore.
    • Cortana: Enable or disable Cortana.
    • Store Updates: Enable or disable automatic store updates.
    • Automatic Updates: Enable or disable automatic Windows updates.
    • Smart Screen: Enable or disable the SmartScreen filter.

    System Repair

    Repair Options

    • DISM: Use the Deployment Imaging Service and Management Tool to repair Windows images.
    • SFC: Use the System File Checker to scan and repair system files.
    • CHKDSK: Use the Check Disk utility to check and repair disk errors.

    Diagnose Options

    • Battery Health: Generate a battery health report.
    • Memory Diagnostic: Run a memory diagnostic to check for memory issues.
    • Disk Optimizations: Perform disk optimizations.
    • Event Viewer: Open the Event Viewer to check system logs.

    System Debloating

    • Uninstall UWP Apps: Easily uninstall unwanted UWP apps to free up system resources.
    • Remove Temporary Files: Clean up temporary files to free up disk space.

    Privacy Enhancements

    Advertising Settings

    • Advertising ID: Enable or disable the use of an advertising ID for personalized ads.
    • Bluetooth Advertising: Enable or disable Bluetooth advertising.
    • News and Interests: Enable or disable news and interests features.
    • Spotlight Features: Enable or disable Windows Spotlight features.
    • Tailored Experiences: Enable or disable tailored experiences based on diagnostic data.
    • Cloud Optimized Content: Enable or disable cloud-optimized content.
    • Feedback Notifications: Enable or disable feedback notifications.

    Telemetry Settings

    • Telemetry Services: Enable or disable telemetry services.
    • Edge Telemetry: Enable or disable telemetry for Microsoft Edge.
    • Visual Studio Telemetry: Enable or disable telemetry for Visual Studio.
    • Nvidia Telemetry: Enable or disable telemetry for Nvidia products.
    • Chrome Telemetry: Enable or disable telemetry for Google Chrome.
    • Firefox Telemetry: Enable or disable telemetry for Mozilla Firefox.
    • Activity Feed: Enable or disable activity feed.
    • Cdp: Enable or disable Connected Devices Platform (CDP).
    • Diagnostics Toast: Enable or disable diagnostic toast notifications.
    • Online Speech Privacy: Enable or disable online speech privacy settings.
    • Location Features: Enable or disable location features.
    • Biometrics: Enable or disable biometric data collection.

    Other Settings

    • Automatic Restart Sign-On: Enable or disable automatic restart sign-on.
    • Handwriting Data Sharing: Enable or disable sharing of handwriting data.
    • Text Input Data Collection: Enable or disable text input data collection.
    • Input Personalization: Enable or disable input personalization.
    • Safe Search Mode: Enable or disable safe search mode.
    • Activity Uploads: Enable or disable activity uploads.
    • Clipboard Sync: Enable or disable clipboard synchronization.
    • Message Sync: Enable or disable message synchronization.
    • Setting Sync: Enable or disable setting synchronization.
    • Voice Activation: Enable or disable voice activation.
    • Find My Device: Enable or disable the Find My Device feature.
    • SMBv1: Enable or disable SMBv1 protocol.
    • SMBv2: Enable or disable SMBv2 protocol.
